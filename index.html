Pengertian Konstruktor & Enkapsulasi

  • Konstruktor: merupakan metode khusus dalam sebuah class yang dipanggil secara otomatis saat objek dibuat. Konstruktor digunakan untuk menginisialisasi nilai awal dari atribut objek, sehingga objek dapat langsung digunakan setelah dibuat.
  • Enkapsulasi: merupakan konsep dalam Object Oriented Programming (OOP) yang digunakan untuk membungkus data (atribut) dan method dalam satu class serta membatasi akses langsung terhadap data tersebut. Dengan enkapsulasi, data dapat dilindungi dari perubahan yang tidak diinginkan dan hanya dapat diakses melalui getter dan setter.

Jenis Constructor di Dart

  • Default Constructor: Constructor bawaan yang otomatis dibuat oleh Dart jika class tidak memiliki constructor sendiri.
  • Named Constructor: Constructor yang memiliki nama tambahan sehingga satu class dapat memiliki beberapa constructor dengan fungsi berbeda.
  • Parameterized Constructor: Constructor yang menerima parameter sehingga nilai atribut dapat diisi saat objek dibuat.
  • Constant Constructor: Constructor yang digunakan untuk membuat objek immutable (tidak dapat berubah setelah dibuat).
  • Factory Constructor: Constructor khusus yang digunakan untuk mengontrol proses pembuatan objek, misalnya mengembalikan objek yang sudah ada atau membuat objek baru berdasarkan kondisi tertentu.

1. Named Constructor

Named constructor memungkinkan sebuah class memiliki lebih dari satu constructor dengan nama berbeda sehingga objek dapat dibuat dengan cara yang berbeda.

DART SOURCE CODE
import 'dart:convert';

class Mobil {
  String? merk;
  int? tahun;

  // Parameterized constructor
  Mobil(this.merk, this.tahun);

  // Named constructor
  Mobil.baru() {
    merk = 'Toyota';
    tahun = 2023;
  }

  // Named constructor dengan parameter
  Mobil.dariJson(Map json) {
    merk = json['merk'];
    tahun = json['tahun'];
  }

  // Override toString for better print output
  @override
  String toString() {
    return 'Mobil{merk: $merk, tahun: $tahun}';
  }
}

void main() {
  print('--- Demonstrasi Class Mobil ---');

  // 1. Menggunakan constructor default (parameterized constructor)
  var mobilA = Mobil('Honda', 2020);
  print('Mobil A (constructor default): $mobilA');

  // 2. Menggunakan named constructor Mobil.baru()
  var mobilB = Mobil.baru();
  print('Mobil B (named constructor .baru): $mobilB');

  // 3. Menggunakan named constructor Mobil.dariJson()
  String jsonString = '{"merk": "Suzuki", "tahun": 2021}';
  Map jsonData = jsonDecode(jsonString);
  var mobilC = Mobil.dariJson(jsonData);
  print('Mobil C (named constructor .dariJson): $mobilC');

  // Contoh lain dengan named constructor Mobil.dariJson()
  Map anotherJsonData = {'merk': 'Mitsubishi', 'tahun': 2022};
  var mobilD = Mobil.dariJson(anotherJsonData);
  print('Mobil D (named constructor .dariJson): $mobilD');

  print('\n--- Akhir Demonstrasi ---');
}

2. Constant Constructor

Constant constructor digunakan untuk membuat objek yang bersifat immutable atau tidak dapat diubah setelah dibuat. Constructor ini menggunakan keyword const dan semua field dalam class harus bertipe final.

DART SOURCE CODE
class Warna {
  final int r, g, b;
  const Warna(this.r, this.g, this.b);

  @override
  String toString() {
    return 'Warna(r: $r, g: $g, b: $b)';
  }
}

void main() {
  const merah = Warna(255, 0, 0);
  const merahLagi = Warna(255, 0, 0); // Menggunakan instance yang sama di memory

  print(merah);
  print(merahLagi);
}

3. Initializer List

Initializer List digunakan untuk menginisialisasi variabel sebelum body constructor dijalankan. Fitur ini sering digunakan untuk menginisialisasi variabel final atau melakukan validasi data.

DART SOURCE CODE
class Titik {
  final double x, y;

  Titik(double x, double y)
    : x = x.abs(),
      y = y.abs() {
    print('Titik dibuat di (${this.x}, ${this.y})'); // Using this.x and this.y to show the absolute values
    
  }
}

4. Public vs Private Variable

Dalam Dart, semua variabel secara default bersifat public, sehingga dapat diakses dari luar class. Untuk membuat variabel menjadi private, digunakan tanda underscore (_) di depan nama variabel. Variabel private hanya dapat diakses dalam file yang sama.

Contoh: nama → public _nama → private

DART SOURCE CODE
class Person {
  String _name;
  int age;
  Person(this._name, this.age);
  
}

void main() {
  Person person = Person("Saskia", 19);
  print(person.age);
  // print(person._name); // error karena private
  
}

5. Getter

Getter adalah method yang digunakan untuk mengambil atau membaca nilai dari variabel private. Dengan getter, programmer tetap dapat mengakses data tanpa membuka akses langsung ke variabel tersebut.

DART SOURCE CODE
class Person {
  String _name;
  int age;
  Person(this._name, this.age);
  
  String get getName1 {
    return _name;
  }
  
  String getName() {
    return _name;
  }
}

void main () {
  Person person = Person("Saskia Mauli Tantria", 19);
  
  print(person.age);
  
  print(person.getName());
  
  print(person.getName1);
}

6. Setter

Setter adalah method yang digunakan untuk mengubah nilai dari variabel private dengan aturan tertentu. Biasanya setter digunakan untuk melakukan validasi data sebelum nilai disimpan.

DART SOURCE CODE
class Person {
  String _name;
  int age;

  Person(this._name, this.age);

  set setName1(String name) {
    _name = name;
  }

  void setName(String name) {
    _name = name;
  }

  String getName() {
    return _name;
  }
}

void main() {
  Person person = Person("Saskia", 19);

  print(person.age);

  person.setName('Mauli');
  print(person.getName());

  person.setName1 = 'Saskia';
  print(person.getName());
}

7. Getter & Setter Validasi

Getter dan setter juga dapat digunakan untuk menambahkan validasi terhadap data.

Misalnya memastikan nilai tidak negatif atau tidak kosong sebelum disimpan dalam atribut class. Dengan cara ini, data yang tersimpan di dalam objek akan tetap valid dan aman.

DART SOURCE CODE
// Nama : Saskia Mauli Tantria
// NIM  : 251410034
// Kelas: SI2A

class Produk {
  String _nama;
  double _harga;

  Produk(this._nama, this._harga);

  String get nama => _nama;
  double get harga => _harga;

  set nama(String value) {
    if (value.length >= 3) {
      _nama = value;
    } else {
      print('Gagal memperbarui nama: Nama harus memiliki minimal 3 karakter.');
    }
  }

  set harga(double value) {
    if (value > 0) {
      _harga = value;
    } else {
      print('Gagal memperbarui harga: Harga harus lebih besar dari 0.');
    }
  }
}

void main() {
  var produk1 = Produk('Monitor Samsung', 3500.0);

  print('--- Informasi Produk Awal ---');
  print('Nama Produk: ${produk1.nama}');
  print('Harga Produk: Rp ${produk1.harga.toStringAsFixed(2)}');

  print('\n--- Memperbarui Produk ---');
  produk1.nama = 'Keyboard Mekanik';
  produk1.harga = 1200.0;

  print('Nama Produk (setelah update): ${produk1.nama}');
  print('Harga Produk (setelah update): Rp ${produk1.harga.toStringAsFixed(2)}');

  print('\n--- Menguji Validasi Setter ---');

  print('Mencoba mengatur nama menjadi "TV"');
  produk1.nama = 'TV';

  print('Nama Produk (setelah gagal update nama): ${produk1.nama}');

  print('Mencoba mengatur harga menjadi -100.0');
  produk1.harga = -100.0;

  print('Harga Produk (setelah gagal update harga): Rp ${produk1.harga.toStringAsFixed(2)}');

  print('Mencoba mengatur harga menjadi 0.0');
  produk1.harga = 0.0;

  print('Harga Produk (setelah gagal update harga): Rp ${produk1.harga.toStringAsFixed(2)}');

  print('\n--- Status Akhir Produk ---');
  print('Nama Produk: ${produk1.nama}');
  print('Harga Produk: Rp ${produk1.harga.toStringAsFixed(2)}');
}

8. Studi Kasus Login

Contoh sistem login sederhana menunjukkan bagaimana enkapsulasi melindungi data username dan password agar tidak bisa diubah secara sembarangan dari luar class.

Proses login biasanya dilakukan melalui method tertentu yang akan memeriksa apakah username dan password yang dimasukkan sesuai dengan data yang tersimpan.

DART SOURCE CODE
class User {
  String _username;
  String _password;

  User(this._username, this._password);

  String get username => _username;

  bool login(String username, String password) {
    return _username == username && _password == password;
  }

  void gantiPassword(String lama, String baru) {
    if (_password == lama) {
      if (baru.length >= 8) {
        _password = baru;
        print('Password berhasil diubah.');
      } else {
        print('Gagal mengubah password: Password baru harus memiliki minimal 8 karakter.');
      }
    } else {
      print('Gagal mengubah password: Password lama tidak sesuai.');
    }
  }
}

void main() {
  print('--- Inisialisasi User ---');

  final User user1 = User('admin_user', 'securePass123');

  print('User created with username: ${user1.username}');
  print('');

  print('--- Uji Coba Login ---');

  String testUsername = 'admin_user';
  String correctPassword = 'securePass123';
  String incorrectPassword = 'wrongPass';

  if (user1.login(testUsername, correctPassword)) {
    print('Login successful for user: $testUsername');
  } else {
    print('Login failed for user: $testUsername');
  }

  if (user1.login(testUsername, incorrectPassword)) {
    print('Login successful for user: $testUsername (This should not happen)');
  } else {
    print('Login failed for user: $testUsername (Incorrect password, as expected)');
  }

  print('');

  print('--- Uji Coba Ganti Password ---');

  print('Attempting to change password from "securePass123" to "newSecurePass456":');

  user1.gantiPassword('securePass123', 'newSecurePass456');

  if (user1.login(testUsername, 'newSecurePass456')) {
    print('Login successful with new password "newSecurePass456".');
  } else {
    print('Login failed with new password (Something went wrong).');
  }

  print('');
}

Kesimpulan

Konstruktor dan enkapsulasi merupakan dua konsep penting dalam Object Oriented Programming. Konstruktor digunakan untuk memastikan objek dibuat dengan nilai awal yang benar, sedangkan enkapsulasi digunakan untuk melindungi data dan mengontrol akses terhadap atribut dalam class.

Dengan menerapkan kedua konsep ini, program menjadi lebih terstruktur, aman, dan mudah untuk dikembangkan.
